Academic Writing, TU Delft, Q3, Group Project

# Repository Contents

# Summary

Conducted a literature review on graph signal processing (GSP) for modelling structural and functional brain connectivity. Surveyed approaches combining GSP with deep learning for neuroimaging analysis and connectivity matrix generation, and assessed their potential for mental disorder diagnosis.
